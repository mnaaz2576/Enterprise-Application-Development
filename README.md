# Enterprise-Application-Development

This repository contains all weekly lab experiments and programs for the **Enterprise Application Development (EAD)** course.
Each week focuses on different web development concepts ranging from basic HTML to full-stack development using modern technologies.

---


## ▶️ How to Run the Programs

### For HTML/CSS/JS Programs

* Open the `.html` file in any web browser

### For React Applications

```bash
npm install
npm run dev
```

### For Node.js / Backend Programs

```bash
npm install
node server.js
```
